# A simple User Profile form

## Stack:
- [React](https://reactjs.org)
- [Ionic](https://ionicframework.com)
- [React Color](https://casesandberg.github.io/react-color/)
- [Day.js](https://day.js.org/)

## Steps:
- Clone repo
- `npm install`
- `ionic serve`
